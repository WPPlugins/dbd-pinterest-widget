This simple Pinterest widget features an easy way to add multiple Pinterest widgets to your website.

== Description ==

The Designer by Design Pinterest widget is highly customisable, featuring the ability to specify a user account link, user account feed and number of items to display within the widget block. The widget works by parsing the Pinterest user feed to provide thumbnails and a short descriptions of each feed item. Cross-browser CSS and JavaScript is used to provide a thumbnail preview of each item within the widget. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the widget using the appearance screen
